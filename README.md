# 2D-flinsim

2D fluid simulations written in C from scratch. This uses my [linear algebra library](https://github.com/brudihawo/la)


## Getting started
Clone the repo
```
$ git clone https://github.com/brudihawo/2d-flinsim.git
```

Update la library
```
$ git submodule init
```

build everything
```
$ make -j$(nproc)
```
